    Version 0.1

    Requires:
        Python 2.7.3
        PyGame 1.9.1
        
    To run:
        cd to the folder.
        python launcher.py
        
    Tested on Lubuntu Linux only.

-------------------------------------------------------------------

 Survive the onslaught of drones to meet
 your nemesis!
 
    Use WASD to move.
    J to fire.
    m to save an in-game screenshot.
    F11 on menu screen to go full screen
    F12 to quit game
    
    Tip:Some drones need hit more than once!
